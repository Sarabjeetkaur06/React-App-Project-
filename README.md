# Task Tracker Application

## Description
This is a simple Task Tracker (To-Do) application built using React.  
Users can add tasks, mark them as completed, and delete tasks.

## How to Run
1. Install Node.js
2. Open terminal in project folder
3. Run:
   npm install
   npm run dev

## Example Usage
- Enter a task name
- Click "Add Task"
- Mark tasks as completed using checkbox
- Delete tasks when finished
